## ğŸ¯ Usage-examples

To use these scripts, you need to have Python installed on your system. You can then run the scripts from the command line using the following command:

```bash
python script_name.py
```

Make sure to replace `script_name.py` with the actual name of the script you want to run. Some scripts may require additional dependencies, which can be installed using pip:

```bash
pip install -r requirements.txt
```

### Examples

1.  **Running the `convertidor.py` script:**

    ```bash
    python convertidor.py
    ```

    Follow the script's prompts to convert files.

2.  **Running the `fusionar_pdf.py` script:**

    ```bash
    python fusionar_pdf.py
    ```

    This script will guide you through the process of merging PDF files.
